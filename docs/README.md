## 当前交付边界

- Windows x64：当前公开支持，提供 Portable EXE 与 ZIP。
- macOS：未来适配，不属于当前公开下载范围，仓库当前不提供自动构建工作流。
- Android：尚未实现，需要单独建立移动端容器。
- 当前 Windows 构建可能尚未完成正式代码签名。
